# PPT Maker - 幻灯片类型参考

## 支持的页面类型

| 类型 | 说明 | 布局特点 |
|------|------|----------|
| `cover` | 封面页 | 全屏深色背景 + 居中标题 |
| `toc` | 目录页 | 左侧色块 + 列表项 |
| `content` | 内容页 | 顶部色带 + 要点列表 |
| `summary` | 总结页 | 左侧色块 + 核心要点 |
| `thank_you` | 致谢页 | 全屏深色 + 居中文字 |

## 三列卡片布局（成果展示）

每项核心成就使用三列卡片展示：

```
┌─────────────────┬─────────────────┬─────────────────┐
│   🟠 背景/痛点   │   🔵 方案/创新   │   🟢 成果/影响   │
│                 │                 │                 │
│  原有流程等待    │  引入智能预约    │  等待时间缩短    │
│  时间超过45分钟  │  系统+诊间结算  │  至18分钟        │
└─────────────────┴─────────────────┴─────────────────┘
```

### 卡片类型定义

| card_type | 用途 | 视觉提示 |
|-----------|------|----------|
| `pain_point` | 背景/痛点 | icon_warning, icon_target |
| `innovation` | 方案/创新 | icon_gear, diagram_workflow |
| `metric` | 成果/量化 | chart_upward_trend, icon_chart |

## 布局规范

- 标准尺寸：10" x 5.625" (LAYOUT_16x9)
- 页码位置：x: 9.3", y: 5.1"
- 字体：中文用 Microsoft YaHei，英文用 Arial
- 颜色格式：6位十六进制，不带 #

## 设计原则

1. **文字精炼** — 幻灯片上字不如表，表不如图
2. **每页3-5要点** — 核心信息不超过5条
3. **视觉层次** — 标题 > 要点 > 细节
4. **一致风格** — 统一配色、字体、间距
