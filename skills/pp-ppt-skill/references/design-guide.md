# PPT Design Guide - 设计规范参考

## 配色系统

### 主色系

| 色系 | 用途 | 色值 |
|------|------|------|
| 蓝色系 | 医疗、企业、科技 | #1F5C99, #4285F4, #0079BF |
| 绿色系 | 医疗健康、环保 | #2E8B57, #43A047, #009688 |
| 橙色系 | 教育、餐饮、活力 | #FF8F00, #F57C00, #E65100 |
| 紫色系 | 创意、艺术、女性 | #7B1FA2, #9C27B0, #6A1B9A |
| 红色系 | 警示、对比、强调 | #D32F2F, #F44336, #C62828 |

### 背景色

- **浅色背景**: #FFFFFF, #F5F5F5, #FAFAFA, #F0F4F8
- **深色背景**: #263238, #37474F, #1A1A2E, #0D1117

### 文字色

| 类型 | 色值 | 用途 |
|------|------|------|
| 主文字 | #212121 / #1A1A1A | 标题、正文 |
| 副文字 | #757575 / #9E9E9E | 副标题、说明 |
| 深色背景文字 | #FFFFFF / #F5F5F5 | 深色背景上的文字 |
| 链接/强调 | #1976D2 / #2196F3 | 链接、重点强调 |

## 字体规范

- **中文**: 微软雅黑、思源黑体、苹方

- **代码/数字**: Consolas, Monaco, Source Code Pro

## 布局规范

### 页面尺寸

- **16:9 (标准)**: 13.333 x 7.5 英寸 (1920 x 1080px)
- **4:3 (传统)**: 10 x 7.5 英寸 (1024 x 768px)

### 安全边距

```
┌─────────────────────────────────────────┐
│  ↑ 0.5 inch                             │
│ → │                               │ ←   │
│   │                               │     │
│   │      内容区域 (Content)       │     │
│   │                               │     │
│ → │                               │ ←   │
│  ↓ │_______________________________│     │
│     ↑ 0.5 inch                     ↓     │
└─────────────────────────────────────────┘
```

### 内容区域

- 左右边距: ≥ 0.5 inch
- 上下边距: ≥ 0.3 inch
- 多列布局列间距: ≥ 0.3 inch

## 幻灯片类型

### 1. 封面页 (Cover)

要素:
- 主标题 (Title)
- 副标题/日期 (Subtitle/Date)
- Logo/装饰元素

布局:
- 居中布局 (Center)
- 左侧标题 (Left Title)
- 全屏背景 (Full Bleed)

### 2. 目录页 (TOC)

要素:
- 页面标题
- 章节列表
- 编号/图标

### 3. 内容页 (Content)

**单栏布局**
```
┌─────────────────────────────────────────┐
│  标题                                    │
│─────────────────────────────────────────│
│                                         │
│  • 要点1                                │
│  • 要点2                                │
│    - 子要点2.1                          │
│    - 子要点2.2                          │
│  • 要点3                                │
│                                         │
└─────────────────────────────────────────┘
```

**双栏布局**
```
┌─────────────────────────────────────────┐
│  标题                                    │
│─────────────────────────────────────────│
│           │           │                  │
│   左侧     │    右侧    │                  │
│   内容     │    内容    │                  │
│           │           │                  │
└─────────────────────────────────────────┘
```

**图文混排**
```
┌─────────────────────────────────────────┐
│  标题                                    │
│─────────────────────────────────────────│
│  ┌──────┐  • 要点1                      │
│  │ 图片 │  • 要点2                      │
│  │      │  • 要点3                      │
│  └──────┘                              │
└─────────────────────────────────────────┘
```

### 4. 对比页 (Comparison)

红绿对比:
- 左侧(负面): 红色系背景
- 右侧(正面): 绿色系背景

蓝橙对比:
- 左侧(传统): 蓝色系
- 右侧(创新): 橙色系

### 5. 数据页 (Data)

要素:
- 图表标题
- 数据可视化 (柱状图、饼图、折线图)
- 数据来源注释

### 6. 结尾页 (Ending)

要素:
- 总结要点
- 感谢语
- 联系方式
- 二维码

强调: #4A90D9 科技蓝

渐变: #667eea → #764ba2

适用: 科技发布会、AI演示、数字产品

## 视觉元素使用

### 图标

- 使用统一样式的图标集
- 建议尺寸: 32x32 - 64x64 px
- 与文字间距: 8-12px

### 形状装饰

- 圆形: 代表完整、循环
- 方形: 代表稳定、结构
- 三角形: 代表方向、箭头
- 线条: 分隔、引导

### 图表

- 柱状图: 比较数量
- 饼图: 显示比例
- 折线图: 展示趋势
- 流程图: 展示步骤

## 常见问题

### Q: 如何选择配色？

A: 根据主题选择
- 企业/商务 → 蓝色系
- 医疗/健康 → 绿色系
- 教育/培训 → 橙色/蓝色
- 科技/未来 → 深色背景 + 亮色点缀

### Q: 内容太多怎么办？

A: 
1. 分成多页
2. 使用双栏布局
3. 提取关键点，减少文字
4. 使用图表可视化

### Q: 如何让PPT更有设计感？

A:
1. 使用统一的配色方案
2. 保持对齐和间距一致
3. 添加装饰元素(线条、色块)
4. 控制每页信息量
5. 使用高质量图片
